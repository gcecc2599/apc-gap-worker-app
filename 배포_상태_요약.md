### 배포 방법

#### 방법 1: Railway (권장)
1. https://railway.app/new 접속
2. GitHub 계정으로 로그인
3. "GitHub Repository" 선택
4. `gcecc2599/apc-gap-worker-app` 선택
5. 자동 배포 시작

#### 방법 2: Render (대안)
1. https://dashboard.render.com 접속
2. GitHub 계정으로 로그인
3. "New +" → "Web Service" 클릭
4. 저장소 연결: `https://github.com/gcecc2599/apc-gap-worker-app`
5. `render.yaml` 자동 감지
6. 배포 시작
