# 🌐 인터넷 배포 가이드

## 현재 상태
- ✅ 반응형 디자인 적용 완료 (모바일, 태블릿, 데스크톱 지원)
- ✅ Netlify CLI 설치 완료
- ✅ 빌드 설정 완료

## 🚀 Netlify 배포 방법

### 1단계: Netlify 로그인
```powershell
cd C:\Users\user\Desktop\webapp
netlify login
```
브라우저가 열리면 Netlify 계정으로 로그인하세요.
- 계정이 없으면 https://app.netlify.com 에서 무료 계정 생성

### 2단계: 프로젝트 초기화
```powershell
netlify init
```
질문에 답변:
- "Create & configure a new site" 선택
- 사이트 이름 입력 (예: `apc-gap-record`)
- 빌드 명령: `npm run build`
- 배포 디렉토리: `dist`

### 3단계: 배포
```powershell
# 프리뷰 배포 (테스트용)
netlify deploy

# 프로덕션 배포 (공개 URL 생성)
netlify deploy --prod
```

### 4단계: 공개 URL 확인
배포 완료 후 다음과 같은 URL이 생성됩니다:
- `https://your-site-name.netlify.app`
- 이 URL은 인터넷 어디서나 접속 가능합니다!

## 📱 반응형 디자인
프로젝트는 이미 반응형으로 설계되어 있습니다:
- **모바일** (< 640px): 터치 친화적 UI
- **태블릿** (640px - 1024px): 중간 크기 레이아웃
- **데스크톱** (> 1024px): 전체 기능 레이아웃

## 🔄 자동 배포 설정 (선택사항)
GitHub에 코드를 푸시하면 자동으로 배포됩니다:
1. GitHub 저장소 생성
2. Netlify 대시보드 → Site settings → Build & deploy
3. "Link to Git provider" 클릭
4. GitHub 저장소 연결

## 📞 문제 해결
- 빌드 실패: `npm run build` 로컬에서 먼저 테스트
- 배포 실패: Netlify 대시보드의 로그 확인
- 도메인 설정: Netlify 대시보드 → Domain settings
